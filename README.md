This end-to-end machine learning project demonstrates the development of a real-time loan approval prediction system, from initial model training to deployment. I utilized a robust model comparison framework involving Logistic Regression, RidgeClassifier, Random Forest, and XGBoost, ultimately achieving an 83% F1 score with the tuned XGBoost model. The system features a FastAPI backend and an interactive Streamlit web application, allowing users to receive instant predictions. To manage the full MLOps lifecycle, I integrated MLflow for comprehensive experiment tracking and DagsHub for seamless model versioning and collaboration.
